## Usage

```jsx "Head"
// HeadExample.jsx
import { Head } from 'jsx-email';

const Email = () => {
  return (
    <Head>
      <title>My email title</title>
    </Head>
  );
};
```

:::tip[Common Usage]
Place metadata, styles, and format detection settings in the `Head` component.
:::

## Props

### TypeScript Interface
```tsx
export interface HeadProps extends BaseProps<'head'> {
  enableFormatDetection?: boolean;
}
```

### Optional Props

#### enableFormatDetection
```ts
enableFormatDetection?: boolean;
```
Controls the `format-detection` meta tag. Defaults to `false`.

### Additional Props

This component expresses all of the [Common Component Props](https://react.dev/reference/react-dom/components/common) for `ComponentProps<'head'>`.
